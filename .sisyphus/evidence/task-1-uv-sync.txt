== Why import-deps fix was needed ==
The upstream rut repo (schettino72/rut) has its own [tool.uv.sources] with:
  import-deps = { path = "/home/eduardo/work/import-deps", editable = true }
When uv resolves rut from git, it reads rut's pyproject.toml and tries to use
that local path as a subdirectory, which fails. The fix: provide static
dependency-metadata for rut so uv skips reading its tool.uv.sources.
